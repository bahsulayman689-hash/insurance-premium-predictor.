# 🏥 Medical Insurance Cost Estimator

An interactive, end-to-end Machine Learning web application that predicts annual and monthly medical insurance premiums based on individual health habits and demographic metrics.

Built with **Python**, **Scikit-Learn**, and **Streamlit**.

---

## 🚀 Live Demo
🔗 **[https://insurance-premium-predictor-4ankzjpv3zpishogphc2yh.streamlit.app/]**

---

## 🎨 Features
* **Center-Stage Form**: Intuitive UI elements for inputting personal metrics (Age, BMI, Children, Sex, Smoking Status) right in the center of the dashboard.
* **Instant Financial Breakdown**: Provides side-by-side estimates for both **Annual Premiums** and broken-down **Monthly Payments**.
* **Risk Tier Indicators**: Automated warning banners highlighting smoker/non-smoker risk levels.
* **Developer Sidebar**: High-contrast red sidebar featuring developer profile assets and direct portfolio connection links.

---

## 🛠️ Tech Stack & Architecture
* **Frontend UI**: Streamlit 
* **Data Processing**: Pandas & NumPy
* **Machine Learning**: Scikit-Learn (`LinearRegression` / `StandardScaler`)
* **Model Serialization**: Joblib

## ⚙️ Local Installation & Setup

Follow these quick steps to get the app running on your machine:

### 1. Clone the Repository
```bash
git clone https://github.com
cd insurance-premium-predictor
```

### 2. Install Dependencies
Make sure you have Python installed, then run:
```bash
pip install -r requirements.txt
```

### 3. Run the Web Application
```bash
streamlit run app.py
```
